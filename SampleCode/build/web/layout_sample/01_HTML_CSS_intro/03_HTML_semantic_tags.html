A newer addition to HTML is the use of "semantics". Instead of having a div with an id like "header", there are several new HTML tag names that you can use (like header, aside, main, footer). These act like divs but just have "better" (more meaningful) names. To style any HTML element (like p or body), you specify the CSS selector with just the HTML name (not starting with # like you would do to style an id).
