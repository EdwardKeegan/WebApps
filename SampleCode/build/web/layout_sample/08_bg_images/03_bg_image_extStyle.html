Here is a third example using a background image -- this time using an external style sheet (the background image url must be specified with respect to the folder where the style sheet is).

Because this image was color, I adjusted the hue (red/green/purple) and saturation (how much color versus black/white) as well as contrast and brightness. I also increased the size of the image. (You can see the original image stored as forest_org.jpg).

It is still important to make sure all text is legible. I increased the line-height and added a little letter-spacing.
