Fixed Positioning (titleNav and footer)

When an element has fixed positioning (as we have done in this example with the titleNav and footer), that element is "removed from the normal flow" and placed wherever you specified (top/bottom, right/left) with respect to the edges of the browser window.

  • This typically places the fixed element on top of (or underneath) some other element. So, you need to use z-index to control which element you want on top (higher value is on top).
  • Then apply padding or margin to make sure that (for example) that the titleNav does not cover up the top of the content and the footer does not cover up the bottom of the content.

Notice also that I made the titleNav and footer both black and the body color is the green showing through the content. You can test this page with very little content. It still looks good - the green color of the content goes all the way down to the footer (because it's actually the color of the body).
