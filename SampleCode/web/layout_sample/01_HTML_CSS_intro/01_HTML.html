Your First HTML Page

HTML stands for HyperText Markup Language. Right click this page, select "View Source", and you'll see what HTML looks like. HTML has a finite list of about 50 elements, most of which have a starting tag and an end tag. For example, a "p" tag represents a paragraph and looks like this: <p> ...the text of the paragraph ... </p>. There are also HTML tags for images, bullet items, tables, and more.

Head Element

In the head element you place information for the browser, things like:

  • the page title (shows up in the tab of your browser),
  • a character set specification (tells the browser which character set you'll use -- UTF-8 instead of ASCII or ANSI),
  • a viewport specification (helpful when designing web pages that look good on desktop as well as mobile viewports), and
  • a <style> element inside which you can specify style rules to make our page look better than just black/white.

Body Element

In the body element, you put the HTML elements that will be seen in the browser window. The body element of this page (right click and View Source) has examples of these HTML tags:

  • "h1" tag, "h2" tag, "h3" tags (different levels of headings),
  • "p" tags (paragraphs),
  • "ul" and "li" tags (bullet items: ul stands for unordered list, li stands for list item),
  • "strong" tags (define where to start/stop bolding), and
