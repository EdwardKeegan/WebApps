Using Flex Box to Place Two Divs on the Same Row

This example puts the title div and the nav div on the same row by assigning CSS declarations "display:flex" and "flex-direction:row" to the titleNav (container of title and nav). Without this styling, the title div and nav div would both start and stop on a new line because divs are block elements that (by default) have that behavior. To make the title align left and the nav align right, use the "justify-content:space-between" CSS declaration (on titleNav, the container).

Narrow the Browser

As always, narrow your browser to see how things change with different viewport widths. What you'll see in this case is that once the title begins to "run into" the nav bar, a horizontal scroll bar kicks in - to allow you to see all the links in the nav bar. This is not ideal. We'll see ways to improve on this when we learn about Responsive Design.
