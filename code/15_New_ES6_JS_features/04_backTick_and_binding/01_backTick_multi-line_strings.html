ES6 Backtick - a third type of quote.

When using JavaScript to create HTML, a lot of quotes are involved. You can use single quotes ' and/or double quotes ", but sometimes you may need three levels of quotes (quotes inside of quotes inside of quotes). That's where the backtick can come in handy. AND with a backtick, you can define multi-line String literals. This is very nice when defining a large block of HTML code as a JavaScript String literal.

Here's the code. 

                                            
    var message = `
        <h1 id="welcome">Welcome</h1>
        <p>We love having you here</p>
    `;
    document.getElementById("output").innerHTML = message;
