Flow Of Control - Press F12 in Chrome to See Console

This example shows how you can intersperse JavaScript function definitions with statements. As you would expect, functions only run if invoked. Statements at the "global" (or page) level, run (top to bottom) at page load time. Here's the code that's being run: 

    console.log('hello');
    function dostuff(x) {
        console.log("x is " + x);
    }
    console.log("goodbye");
    dostuff("later");
