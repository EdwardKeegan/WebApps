Templating with ES6 Backtick and Binding. Press F12 (open JS console) to see output.

With ES6, you can place something like

$(varName)

inside of a backtick delimited string. The value of the variable will be placed into the string. Using backtick with binding makes it easy to create HTML templates.

Here's the code: 

    var personName = "sally";
    var dept = "CIS";
    var message = `${personName} is an instructor in the ${dept} department`;
    console.log(message);
    document.getElementById("output").innerHTML = message;
