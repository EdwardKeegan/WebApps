HTML Tags For Getting Input

This page shows how web applications USED TO use <form> tags to get and process user data, e.g., access a database.

  • There are three <form> tags on this page, each containing HTML input elements and a <submit> button.
    • Each form is styled with a box around it so that it is easy for you to see where the forms are on the page.
  • Each form has an "action" attribute that specifies the names of the server side program that is to "process" the users input -- when the user clicks submit. 
        <form method="post" action="http://cis-linux2.temple.edu:8080/sallyk/receiveData.jsp">

    "receiveData.jsp" (shown in the form's action attribute above) is the server side program (or page) that is referenced by all three forms on this page. I wrote and published this generic program just to demonstrate how web pages post data to a server side program. All my page does is to display the names and values of all the inputs that were sent to it. BUT, keep in mind, that server side programs can do much more (like access a database which is very important for web applications).
