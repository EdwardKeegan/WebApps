Function is a Data Type: Click F12 to See Console Output

In JavaScript, you don't have to specify data type when you declare a variable. You just use the keyword "var". The type of a variable depends on what value happens to be in the variable and that can change over time (as code example shows).

In JavaScript, a function is a first class data type, just like integer or real number or string.

Here's the code that's being run: 

    var x = 5;
    console.log(x); // prints 5
    x = "sally";
    console.log(x); // prints sally
    function myFunc(y) {
        console.log(y + 1);
    }
    x = myFunc;
    x(50); // prints 51 (invokes myFunc, passing in 50).
