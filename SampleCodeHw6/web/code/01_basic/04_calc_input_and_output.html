Adding Calculator - 3 Versions (3 grey buttons)

  • Version 1: doesn't convert to numeric, concatenates instead of adding. 
        function add_v1() { // this version does not convert the input values to numbers, so the result is concatenation, not addition.
        document.getElementById("answer").innerHTML =
                document.getElementById("leftside").value +
                document.getElementById("rightside").value;
    }
  • Version 2: converts to numeric then adds. If either text box holds a non-number, the answer is NaN (stands for Not a Number). 
        function add_v2() { // this version converts to numbers then adds, but it doesnt check to make sure the values are valid numbers.
        document.getElementById("answer").innerHTML =
                Number(document.getElementById("leftside").value) +
                Number(document.getElementById("rightside").value);
    }
  • Version 3: checks that both values are numbers, converts, then adds. 
        function add_v3() { // this version checks that the user input is a number "NaN" means "Not a Number"
        if (isNaN(document.getElementById("leftside").value)) {
            alert("Silly - if you don't put a number in the left side, how can I add up these numbers for you?");
            return;
        }
        if (isNaN(document.getElementById("rightside").value)) {
            alert("Duh - you have to put a number in the right side too !");
            return;
        }
    }
