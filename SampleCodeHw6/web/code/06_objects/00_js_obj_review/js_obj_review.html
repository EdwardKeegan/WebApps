This code example reviews what we know about JavaScript objects:
  • That you can create an object "on the fly" with all object properties being public.
  • Any object property can be a function, just like any other data type.
  • You can dynamically add properties to an object.
  • You can console.log an object and this shows you all the properties of the object and their values.
  • You can access an object's property using associative array notation.
  • You can iterate over all the properties in an object and discover the property names and values.
