This code is perhaps the simplest way to use JS to implement a drop down menu, but it has some problems. The "HTML coder" has to assign ids to all the drop content divs and they have to add a click event to all the drop header divs. Also, this code also violates software dependency injection because it presumes that there are two CSS class names (that were NOT passed in) - "show" and "hide". Finally, if the user clicks somewhere outside of the nav area, the drop downs do not close (and you can open more than one drop down menu at the same time).
