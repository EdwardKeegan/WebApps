The parent (grey div) is position:relative. Relative positioned elements can be "jogged" up or down, left or right a little -- from where they would have been, but we did not add any left/right, top/bottom attributes to the grey parent, so the parent acts pretty much like any other (default/static) div.

The pink, yellow, and blue children are position:absolute.

  • Absolutely positioned elements are removed from the normal flow, then (depending on top/bottom left/right) placed with respect to the edges of their first non static parent.
  • Note that the absolute children overwrite the content of the parent. If you didn't want this, you'd add margin or padding (like we added padding to the top of the content so it was not hidden by the fixed titleNav).

Absolutely positioned children are great for drop down menu content because the size of the drop down menu content does not impact the size of the parent (dropHeader), especially as the drop content becomes visible/invisible.
