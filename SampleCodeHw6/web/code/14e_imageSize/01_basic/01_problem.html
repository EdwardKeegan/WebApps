Problem: Can't Tell Size of <img> Until It's On The Page

Open Chrome's Console To See Output

In this page, we are trying to create an <img> tag using JS and then we want to know the size of the image (e.g., in pixels) right away. We may need to know the actual (pixel) image size for various "effects" like image zoom.

The problem is that the size of the image (especially if being set from a CSS style, possibly as a percentage) will not be known until after the <img> has been added to the page. BUT the image is not added to the page until after our "Make" function creates and returns the component with the image in it.
