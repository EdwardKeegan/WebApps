ES6 Function Parameter Using Object Destructuring. Press F12 (open JS console) to see output. 

    function demo ( {a, b} ) {
        console.log("a is " + a + " and b is " + b); // prints a is apple and b is banana
    }

    // putting the properties in reverse order to show the association is by 
    // property name, not position.
    demo( { b: "banana", a: "apple"} );
