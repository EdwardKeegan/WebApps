Home

Here is how a web designer (with no ability to program) might make the Navigation Bar work. The web designer could copy index.html, rename it (e.g, blog.html), then edit the content area of the new page. Links within the Navigation Bar would reference these pages.

Click on the two working links ("home" and "blog") to see how this simple navigation would work.

The problem with this approach is that there is no reuse of the User Interface. If you had a 500 page web site and you wanted to modify the Nav Bar, you'd have to edit each of the 500 pages.
