Fixed Column Headings in HTML Table

To test this code, scroll up and down within the data rows of the HTML table (which should show a vertical scroll bar to its right). When you do this, you should see the column headings do not disappear. This styling is especially helpful in Mobile Layout (where screen space is limited), but it is also helpful in Desktop Layout when there are a lot of data rows.

Look for /* NEW */ in the clickSort.css to see the relevant styling rules:

  • The first thing you do is make the container of the HTML table be a certain height (it's set to 20rem in the example, but make it what you want), then set "overflow: auto" which enables scrolling behavior within the HTML table container.
  • Then you just have to style the <th> tags to be "position:sticky" (rather like absolute behavior relative to the HTML table container, keeps them always at the top). Also, make sure you specify "top:0" (so that the column headers "stick" to the top) and a z-index value that's high enough (so that the column headers are on top of the HTML table's data rows).
