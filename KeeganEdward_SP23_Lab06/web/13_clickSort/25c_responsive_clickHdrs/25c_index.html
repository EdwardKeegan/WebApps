Header Click Events with Responsively Designed HTML Table

To test this, click on column headings (both in Desktop as well as Mobile Layouts). I have added a "dummy" onclick event that just pops up a "snackbar" (message that closes itself after a couple of seconds). I have provided this example just to show how you can add onclick events to column headings, whether you are in Mobile or Desktop Layout.

When you implement your homework, you will not need to use the "snackbar". I only used the snackbar so that you would not have to close all the messages when you test click each column heading.
